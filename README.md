# thesis_summarization
OpenAPI를 이용한 논문, 기사, 글 요약 및 번역
## TLDRThis
### 1. Abstractive(Human-like) summarization
 * Abstractive summarization(생성 요약)은 기존 Input text를 그대로 인용하지 않고, 기존의 내용을 새롭게 re-pharasing 하여 Summary를 생성하는 요약모델
### 2. Extractive summarization
 * Extractive summarization(추출 요약)은 기존 Input text에 존재하는 중요한 단어를 그대로 사용하여 Summary를 생성하는 요약 모델
